# Task Manager - Prueba Técnica Becario

## Descripción

Se desarrolló una aplicación web de gestión de tareas utilizando React.

Dicha aplicación permite:
- Crear nuevas tareas
- Visualizar tareas
- Marcar tareas como completadas
- Eliminar tareas

Se implementó persistencia utilizando LocalStorage, lo que permite mantener la información incluso al recargar la página.

---

## Herramientas implementadas

- React
- JavaScript
- HTML/CSS
- LocalStorage

---

## Instrucciones de ejecución

1. Clonar el repositorio
2. Ejecutar:
   npm install
3. Ejecutar:
   npm start
4. Abrir en el navegador:
   http://localhost:3000

---

## Decisiones técnicas tomadas

- Se utilizó React por su enfoque basado en componentes
- useState para manejar el estado de las tareas
- useEffect para persistencia de datos
- LocalStorage para evitar backend
